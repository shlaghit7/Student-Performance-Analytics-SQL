# 🎓 Student Performance Intelligence System

## 🚀 Overview

This project is an end-to-end data analytics system designed to track student performance, attendance, and identify at-risk students using SQL, Power BI, and Machine Learning.

## 📊 Features

* Student Performance Tracking
* Attendance Analysis
* At-Risk Detection System
* KPI Dashboard (Power BI)
* ML-based Risk Prediction

## 🛠 Tech Stack

* SQL (PostgreSQL)
* Excel
* Power BI
* Python (Machine Learning)
